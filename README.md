## 👨‍💻 About Me

I am a **Technical Analyst at Cognizant Technology Solutions**, working on a Google client engagement involving large-scale JSON and XML data feed validation, API pipeline management, and systematic QA operations. My production exposure spans structured data contracts, ingestion reliability engineering, and cross-functional coordination across data and product teams.

Outside the day job, I build independently — shipping full-stack MERN applications, AI-powered tooling integrating LLM and voice APIs, and backend systems with a focus on API design, pipeline integrity, and developer experience. I approach every project with a **product engineering mindset**: reliability, observability, and clean interfaces over complexity for its own sake.

- **Backend Engineering:** RESTful API design, data pipeline architecture, Express.js, Node.js, MongoDB — with production-grade debugging experience at Cognizant on a Google client account.
- **AI/ML Integration:** LLM API orchestration via LangChain, generative AI chatbots, voice AI agents using ElevenLabs and Twilio, prompt engineering for structured outputs.
- **Full-Stack Development:** MERN stack end-to-end — React frontends, Express REST APIs, MongoDB Atlas, JWT authentication, role-based access control.
- **Product Engineering:** Systematic QA, data contract validation, clean code practices, and a bias toward shipping working software over perfect architecture.

**Open To:** Entry-Level SDE / Full-Stack / Backend Developer / AI Engineer roles — Hyderabad, Bengaluru, or Remote.
